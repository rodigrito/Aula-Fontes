# Atividade de CSS

Repositório dedicado à nova atividade prática de desenvolvimento web do IOS. O objetivo principal deste projeto é aplicar fundamentos de estilização com fontes em uma página, respeitando a restrição de propriedades CSS estabelecida para a entrega.

## 🎨 Propriedades CSS:

A estilização deste projeto foi construída utilizando **exclusivamente** as seguintes propriedades:

- `font-family`: Para definição da família de fontes da página.
- `font-style`: Para definir o estilo da fonte de um texto, permitindo alternar entre três valores principais: normal, italic (itálico) ou oblique.
- `font-size`: Para o dimensionamento da tipografia.
- `display`: Para definir como um elemento HTML é renderizado e disposto na página (se comporta como bloco, linha, etc.), controlando o layout e a visibilidade.

## 💻 Tecnologias Utilizadas

- HTML5
- CSS3
